# April News for Dallas.rb

### Requirements

* Jekyll (ruby)

### How to view

1. Clone the repository
2. `cd` to the directory
3. `script/bootstrap`
4. `script/server`
5. Open [localhost:4000](http://localhost:4000) in your favorite browser
6. Navigate using arrow keys
